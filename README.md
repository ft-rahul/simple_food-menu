**Reestar Restaurant Website**

Reestar is a simple restaurant website developed with just HTML and CSS. This project demonstrates a clean design for a food menu with beautiful food card design and a simple contact form design. In each food menu, an image, a brief description, and an "Order Now" button are included, mimicking a simple food ordering system.

This project is for practicing basic web development concepts like web page structure, design, image usage, and form creation, among others. This project is a simple exercise for anyone who wants to learn how HTML structures a web page and how CSS styles a web page.

In this website, three main components are included: a header with a restaurant name, a menu section with food items designed in a card style, and a contact section where users can input their name, age, and email for a simple message sending feature.

**Technologies Used: HTML5 and CSS3**
